== Description ==

Integrate your favorite Contact Forms, store submitted DB & SMS notification - Contact Form 7, Gravity Form, Ninja Forms and design it as you want using Elementor page builder. No coding required!
[youtube https://youtu.be/fIk6bww58nE]


### Features

* Pick CF7, Gravity Form, Ninja Forms and style within Elementor page builder
* STORE All your Form Submisssions! - never miss any submission.
* Export the submissions data with just 1 click!

### Installation

Note : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.


1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Can I use the plugin without Elementor Page Builder? =

No. You cannot use without Elementor since it's an addon for Elementor.
